# Worklog

## 2026-06-15: 回答控えメールの回答日時と回答順を調整

- `backend/src/Infrastructure/Mail/MailService.php` の回答日時を `Y/m/d H:i:s` 表記に変更した
- `answer_json` の列挙順ではなく、`questions_json` の設問順で回答内容を並べるようにした
- `questions_json` にしか存在しない余剰キーは、従来どおり末尾に残すようにした

## 2026-06-11: デプロイ時の環境変数と権限まわりの対応

- フロントエンドのビルド用環境変数は、Vite の build 時に参照される値を調べたうえで、デプロイ時に適用する運用にした
- バックエンドの環境変数は、結局 SSH で接続して手動作成した
- Basic 認証のパスワードファイルの配置先を変える必要があったため、`.htaccess` は既存ファイルをコピーせず、配置先を反映した内容をそのまま流し込む形にした
- ステージング環境から `rsync` するとパーミッションが `700` になり、プロセスから読めなかったため、`711` に変更した

## 2026-07-02: 管理者ログインの自動復帰導線

- `AdminShell` 配下の保護ルートは、未認証時に `/manage/login?from=...` へ共通リダイレクトする
- `/manage/login` では LINE 認証後に `POST /api/manage/login` を自動実行し、管理者セッションを作成してから元の admin ルートへ戻す
- これにより、`/manage/surveys` や `/manage/surveys/new` などへの直アクセスでも、手動で二度ログイン操作をする必要がなくなった
